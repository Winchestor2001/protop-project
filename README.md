# 100+ KASB — Premium Professionals Platform

Платформа для связи клиентов и специалистов по 100+ профессиям.

## 📋 Возможности

- **Для клиентов**: Поиск специалистов по профессиям, просмотр профилей, связь через телефон/email
- **Для работников**: Регистрация в системе, добавление своего профиля с указанием опыта, цены, времени работы
- **100+ профессий**: IT, Дизайн, Маркетинг, Бизнес, Образование, Здоровье, Право, Медиа и другие
- **Многоязычность**: Узбекский, Русский, Английский
- **Тёмная тема**: Переключение между светлой и тёмной темами

## 🚀 Установка и запуск

### Шаг 1: Установка зависимостей Python

```bash
pip install -r requirements.txt
```

### Шаг 2: Запуск backend сервера

```bash
python app.py
```

Сервер запустится на http://localhost:5000

### Шаг 3: Открытие фронтенда

Откройте в браузере:
- `index.html` - главная страница
- `kliyent.html` - страница для клиентов
- `ischilar.html` - страница для работников
- `specialists.html` - страница со списком специалистов

## 🔧 API Endpoints

### GET /api/specialists
Получить список всех специалистов или отфильтрованных по профессии

Параметры:
- `profession` (optional) - фильтр по профессии

Пример:
```
GET /api/specialists?profession=Frontend%20Dasturchi
```

### POST /api/specialists
Добавить нового специалиста

Body (JSON):
```json
{
  "profession": "Full Stack Dasturchi",
  "full_name": "Alisher Karimov",
  "phone": "+998901234567",
  "email": "alisher@example.com",
  "experience": 5,
  "price": "50$",
  "free_time": "Dushanba-Juma, 9:00-18:00",
  "description": "Tajribali Full Stack dasturchi..."
}
```

### GET /api/specialists/{id}
Получить данные конкретного специалиста

### GET /api/professions
Получить список всех уникальных профессий

### GET /api/stats
Получить статистику платформы

## 💡 Как использовать

### Для клиентов:

1. Откройте `index.html`
2. Нажмите "MEN KLIYENT"
3. Выберите нужную профессию
4. Просмотрите список специалистов
5. Нажмите "Bog'lanish" для получения контактов

### Для работников:

1. Откройте `index.html`
2. Нажмите "MEN ISHCHIMAN"
3. Выберите свою профессию
4. Нажмите "Tanlash"
5. На странице специалистов нажмите "O'zimni qo'shish"
6. Заполните форму регистрации
7. После добавления ваш профиль появится в списке

## 👨‍💻 Разработка

Для разработки:

1. Запустите Flask в режиме debug (уже включен в app.py)
2. Откройте HTML файлы в браузере
3. Изменения в HTML/CSS/JS отображаются сразу после обновления страницы
4. Изменения в Python требуют перезапуска сервера (или используется auto-reload)

## ⚠️ Важные примечания

- Backend должен быть запущен для работы функций добавления/просмотра специалистов
- Убедитесь, что порт 5000 свободен
- База данных создается автоматически при первом запуске
- Тестовые данные добавляются автоматически
